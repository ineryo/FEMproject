Detailed Description

template<class numericType, unsigned int nNodePerElem, unsigned int spaceDimension, unsigned int nInterpFunOrder>
class baseElemIsoparClass< numericType, nNodePerElem, spaceDimension, nInterpFunOrder >

A base class created with the intention to resume Isoparametric Element functionalities. This class is pure virtual (abstract).

nNodePerElemnumber of nodes in an element.
spaceDimensionthe spatial dimension of this project.
